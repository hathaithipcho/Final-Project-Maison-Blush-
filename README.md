Maison Blush — Perfume E-Commerce Website

"Scent is the memory we choose to create."

เว็บไซต์จำลองร้านค้าน้ำหอม Maison Blush พัฒนาขึ้นเพื่อเป็นส่วนหนึ่งของโครงงานในรายวิชา ICT12267 คณะเทคโนโลยีสารสนเทศ มหาวิทยาลัยศรีปทุม ปีการศึกษา 2568

ภาพรวมโปรเจกต์
Maison Blush เป็นเว็บไซต์ E-Commerce จำลองสำหรับร้านน้ำหอม ออกแบบในสไตล์ Luxury Botanical โทนสีชมพูพาสเทลและเขียวธรรมชาติ มีระบบตะกร้าสินค้า ระบบล็อกอิน รีวิวสินค้า และหน้า Customer Support ครบครัน

⚠️ หมายเหตุ: เว็บไซต์นี้เป็นผลงานเพื่อการศึกษาเท่านั้น ไม่มีวัตถุประสงค์เชิงพาณิชย์ และไม่ได้มีการจำหน่ายสินค้าหรือให้บริการจริงแต่อย่างใด

หน้าต่างๆ ในเว็บไซต์
ไฟล์หน้าคำอธิบาย 
index.html Home หน้าแรก Hero Section พร้อม Animated Petal Canvas 
collection.html Collection รายการสินค้าน้ำหอมทั้งหมด 23 รายการ 
about.html About ข้อมูลเกี่ยวกับโปรเจกต์
contact.html Contact ฟอร์มติดต่อและช่องทางการติดต่อ
reviews.html Reviews รีวิวสินค้าจากลูกค้า พร้อมระบบให้คะแนน
support.html Customer SupportFAQ และช่องทาง Support 
team.html ผู้จัดทำข้อมูลสมาชิกในทีม
login.html and Register.html Login / Register ระบบเข้าสู่ระบบ
