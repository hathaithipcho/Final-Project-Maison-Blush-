เว็บไซต์จำลองร้านค้าน้ำหอม Maison Blush พัฒนาขึ้นเพื่อเป็นส่วนหนึ่งของโครงงานในรายวิชา ICT12267 คณะเทคโนโลยีสารสนเทศ มหาวิทยาลัยศรีปทุม ปีการศึกษา 2568

ภาพรวมโปรเจกต์
Maison Blush เป็นเว็บไซต์ E-Commerce จำลองสำหรับร้านน้ำหอม ออกแบบในสไตล์ Luxury Botanical โทนสีชมพูพาสเทลและเขียวธรรมชาติ มีระบบตะกร้าสินค้า ระบบล็อกอิน รีวิวสินค้า และหน้า Customer Support ครบครัน

⚠️ หมายเหตุ: เว็บไซต์นี้เป็นผลงานเพื่อการศึกษาเท่านั้น ไม่มีวัตถุประสงค์เชิงพาณิชย์ และไม่ได้มีการจำหน่ายสินค้าหรือให้บริการจริงแต่อย่างใด
